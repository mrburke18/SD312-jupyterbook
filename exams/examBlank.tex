\question[4] Describe the process that takes a java source file from a .java
file to execution.

\fbox{\parbox{6in}{\vspace{2in}\ }}
\begin{solution}
The source file is first compiled to bytecode.  During execution, an
interpreter then turns this bytecode into machine code line by line.
\end{solution}
